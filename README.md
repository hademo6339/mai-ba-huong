# Mai Bá Hương Landing Page

Giao diện giới thiệu doanh nghiệp Mai Bá Hương được xây bằng React + Vite + Tailwind CSS, hoàn toàn chạy phía client và không phụ thuộc vào dịch vụ bên ngoài.

## Yêu cầu

- Node.js 18+

## Cách chạy

1. Cài dependency: `npm install`
2. Chạy môi trường dev: `npm run dev`
3. Build production (tùy chọn): `npm run build`

## Ghi chú

- Font Manrope và Playfair Display được bundle nội bộ thông qua `@fontsource`, không cần tải từ CDN ngoài.
